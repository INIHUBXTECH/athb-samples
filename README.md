# 이니허브 통합인증 연동 샘플
이니허브 통합인증 서비스는 다양한 인증수단을 하나로 통합하여 제공하는 서비스 입니다.  
이용기관은 이니허브에서 제공하는 통합인증창을 사용하여 다양한 인증 서비스를 쉽고 간편하게 사용할 수 있습니다.

## 데모

* PHP
* Node.js
